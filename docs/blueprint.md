# **App Name**: Showcase Sphere

## Core Features:

- Project Carousel: Showcase 4 main projects with key details and a visual carousel.
- Contributor Display: Display contributors with avatars that collapse/expand to show names and bios.
- Mentor Spotlight: Highlight the mentor with a photo and a detailed description of their role.
- Animated Background: Integrate particle.js and/or dnajs to create an interactive background.
- Dynamic Landing Page: Design a dynamic landing page.
- Responsive Footer: Implement a clean footer with copyright information and key links.
- Navigation Bar & Landing Page Quote: A proper navigation bar along with a landing page written a quote regarding code

## Style Guidelines:

- Primary color: Soft glacier blue (#94BDBE) to establish a calm, professional atmosphere.
- Background color: Light, desaturated blue (#E2EBEB), creating a subtle backdrop.
- Accent color: Dusty blue (#5E819D), to complement the glacier aesthetic, while providing necessary contrast for callouts and important elements.
- Body text: 'Inter', a sans-serif font for clear readability in descriptions and bios.
- Headline font: 'Space Grotesk', a sans-serif font that has a scientific/tech feel for titles.
- Use clean, minimalist icons for project categories and contributor roles.
- Cards: Design glacier themed cards with good drop shadows and color accents
- Implement subtle transitions and hover effects to enhance user interaction and provide visual feedback.
